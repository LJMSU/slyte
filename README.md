## Install

Please make sure that `youtube-dl` is available in the command line.

```sh
npm install -g slyte
```

### Dependencies

- [`youtube-dl`](https://youtube-dl.org/)
- [`ffmpeg`](https://www.ffmpeg.org/)
- `atomicparsley`

## Usage

```sh
# 1. Display help.
slyte help

# 2. Download a youtube video.
slyte install https://youtu.be/U0TXIXTzJEY
```
